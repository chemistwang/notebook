
## 配置命令行打开vscode

1. 在vscode中输入command+shift+p，打开命令面板;
2. 输入`Shell Command`;
3. 此时会有提示`Shell Command: Install 'cod' command in PATH`，运行即可;
4. 关闭vscode，直接在终端中输入`code .`，则打开vscode
