## CSRF

定义：`Cross-site request forgery`，跨站请求伪造

原理：利用 `网站` 对 `用户浏览器` 的信任

漏洞：简单身份验证只能保证请求发自某个`用户的浏览器`，不能保证请求本身是用户自愿发出的

过程：诱导进入第三方网站，向被攻击的网站发送跨站请求。利用已经获取的注册凭证，绕过后台
