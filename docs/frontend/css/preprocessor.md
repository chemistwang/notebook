### 特性

- `Autoprefixer`: 浏览器前缀
- `PostCSS Preset Env`: 相当于 `JS` 中的 `babel`,可以将新特性语法降级


### 用法

- `webpack`: 配合打包工具使用
- `postcss-cli`: 官网工具
- `postcss api`: 调用api自己实现编译
