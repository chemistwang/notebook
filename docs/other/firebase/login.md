# 登陆

## 注册

访问 [Firebase官方网站](https://firebase.google.com/) 登陆即可


## 命令行工具安装

``` bash
# 安装Firebase命令行工具
sudo npm install -g firebase-tools
# 确认版本号
firebase --version
# Firebase登陆
firebase login
# 列出当前工程一览
firebase projects:list
```

:::warning 登陆问题
如果出现登陆失败的问题，直接使用 `ClashX Pro` 并开启 `Enhanced Mode(增强模式)`
:::
